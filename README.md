# Food_Saver

An iOS SwiftUI app inspired by Too Good To Go, allowing food businesses to offer discounted surplus food to users.

## Features

- Customer and store roles
- Food offer listings
- Reservations
- Pickup confirmation code
- Map-based offer browsing
- Firebase Authentication and Cloud Firestore

## Technologies

- Swift
- SwiftUI
- Firebase
- Cloud Firestore
- MVVM
